When changing CellPanel state to State.OBSTACLE check CellPanel neighbors and color it accordingly.
For each State.OBSTACLE neighbor, check their neighbors and redraw CellPanel.

When holding SHIFT and pressing RIGHT_MOUSE_BUTTON, set CellPanel state to State.OBSTACLE.
When holding SHIFT and pressing LEFT_MOUSE_BUTTON, set CellPanel state to State.EMPTY.
When not holding SHIFT and pressing LEFT_MOUSE_BUTTON, JPupUp apperas with Set State.START or Set State.END.
